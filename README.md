## 🛠️ 技术栈

- [VitePress](https://vitepress.dev/) – 极简静态网站生成器

- [Vue 3](https://vuejs.org/) – 前端框架（VitePress 底层依赖）
- [GitHub Pages](https://pages.github.com/) – 静态页面托管平台
